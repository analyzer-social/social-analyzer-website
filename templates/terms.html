📜 شروط الخدمة

📅 آخر تحديث:

باستخدامك لبوت Social Analyzer (@Social_Media_analyzer_bot)، فإنك توافق على الالتزام بشروط الخدمة هذه.

1. قبول الشروط

باستخدامك لبوت Social Analyzer، فإنك توافق على الالتزام بشروط الخدمة هذه. إذا كنت لا توافق على أي جزء من هذه الشروط، يرجى عدم استخدام البوت.

2. وصف الخدمة

يقدم البوت خدمات تحليل حسابات التواصل الاجتماعي (يوتيوب، تيك توك، انستقرام، فيسبوك، سناب شات) بما في ذلك:

  • 📺 تحليل قنوات يوتيوب (المشتركين، الفيديوهات، المشاهدات، أوقات النشر)
  • 📸 تحليل حسابات سناب شات (المتابعين، مشاهدات القصص، معدلات التفاعل) - عبر Snapchat Kit API
  • 🎵 تحليل حسابات تيك توك (الملف الشخصي، الإحصائيات، الفيديوهات) - عبر TikTok API
  • 📘 تحليل حسابات فيسبوك - قيد التطوير
  • 📷 تحليل حسابات انستقرام - قيد التطوير
  • 📄 إنشاء صفحة بايو شخصية للمستخدمين المميزين

3. تكامل تيك توك (TikTok API)

🎵 شروط خاصة باستخدام ميزة تيك توك:

  • ميزة تحليل تيك توك تتطلب موافقة صريحة منك عبر OAuth 2.0
  • نحن نحصل فقط على البيانات العامة للملف الشخصي وإحصائيات الأداء الأساسية
  • لا نصل أبداً إلى: الرسائل الخاصة، المحادثات، أو المحتوى غير العام
  • يمكنك إلغاء صلاحية التطبيق في أي وقت من إعدادات تيك توك
  • نلتزم بسياسات واستخدامات TikTok API

4. تكامل سناب شات (Snapchat Kit)

📸 شروط خاصة باستخدام ميزة سناب شات:

  • ميزة تحليل سناب شات تتطلب موافقة صريحة منك عبر OAuth 2.0
  • نحن نحصل فقط على البيانات العامة للملف الشخصي وإحصائيات الأداء الأساسية
  • لا نصل أبداً إلى: الرسائل الخاصة، المحادثات، أو المحتوى غير العام
  • يمكنك إلغاء صلاحية التطبيق في أي وقت من إعدادات سناب شات
  • نلتزم بسياسات واستخدامات Snapchat Kit API

5. 🔗 الروابط الخارجية وخدمات فحص اليوزرنيم

📌 تنبيه خاص بخدمة فحص اليوزرنيم (Username Check)

يقدم بوتنا زراً أو رابطاً مختصراً لموقع instantusername.com بهدف تسهيل الوصول إلى خدمة فحص توافر اليوزرنيم على منصات متعددة.

  • 🔍 طريقة الفتح: يتم فتح الرابط في المتصفح الداخلي لتطبيق تلجرام (In-App Browser) وليس عبر خوادم البوت.
  • 🤖 دور البوت محدود: البوت لا يشارك في عملية البحث، ولا يقوم بتحميل النتائج، ولا يخزن أي بيانات تدخلها في الموقع الخارجي.
  • 🌐 استقلالية البيانات: عند فتح الرابط، ينتقل اتصالك مباشرة إلى موقع instantusername.com، وتصبح جميع بياناتك وبيانات البحث خاضعة تماماً لـ سياسة الخصوصية وشروط الخدمة الخاصة بالموقع الخارجي.
  • ⚖️ إخلاء مسؤولية: بوت Social Analyzer ومطوروه غير مسؤولين عن توفر الخدمة، دقة النتائج، أو أي ضرر ناتج عن استخدام موقع instantusername.com. كما أن البوت لا يتحمل أي مسؤولية عن كيفية تعامل الموقع الخارجي مع بياناتك.
📖 ملخص للشفافية: الرابط يُفتح داخل تلجرام مباشرة (متصفح داخلي) -> أنت من يتفاعل مع الموقع -> البوت لا يعرف ماذا تبحث ولا يخزن شيئاً. استخدامك للموقع يعني موافقتك على سياسات instantusername.com.

6. حسابات المستخدمين

أنت مسؤول عن الحفاظ على سرية بيانات حسابك في تلجرام. جميع الأنشطة التي تحدث عبر حسابك هي مسؤوليتك وحدك.

7. الاستخدام المقبول

أنت توافق على استخدام البوت فقط للأغراض القانونية وبما يتوافق مع قوانين بلدك. لا يجوز استخدام البوت لنشر محتوى غير قانوني أو ضار، أو محاولة استخراج البيانات بطرق غير مصرح بها.

8. حدود المسؤولية

لن يكون مطور البوت مسؤولاً عن أي أضرار مباشرة أو غير مباشرة ناتجة عن استخدام أو عدم القدرة على استخدام البوت، بما في ذلك فقدان البيانات أو الأرباح.

9. خدمات الطرف الثالث

يعتمد البوت على واجهات برمجة تطبيقات (APIs) من:

  • Google YouTube API v3 - لتحليل قنوات يوتيوب
  • TikTok API - لتحليل حسابات تيك توك
  • Snapchat Kit API - لتحليل حسابات سناب شات
  • Google Gemini API - لتوصيات الذكاء الاصطناعي

أنت توافق على الالتزام بشروط استخدام هذه الخدمات عند استخدام البوت. بالإضافة إلى ذلك، أي استخدام لخدمات فحص اليوزرنيم عبر الروابط الخارجية يخضع لشروط الموقع المُقدم للخدمة.

10. التعديلات على الخدمة

نحتفظ بالحق في تعديل أو إيقاف الخدمة في أي وقت دون إشعار مسبق. سنبذل جهداً معقولاً لإخطار المستخدمين بأي تغييرات جوهرية.

11. التعديلات على الشروط

قد نقوم بتحديث شروط الخدمة هذه من وقت لآخر. سنقوم بإخطارك بأي تغييرات جوهرية عبر البوت أو عبر قناة التحديثات. استمرار استخدامك للبوت بعد التغييرات يعني موافقتك على الشروط المحدثة.

12. إنهاء الخدمة

نحتفظ بالحق في إنهاء أو تعليق وصولك إلى البوت فوراً، دون إشعار مسبق، لأي سبب كان، بما في ذلك انتهاك شروط الخدمة.

13. قانون التحكيم

تخضع هذه الشروط وتفسر وفقًا لقوانين الجمهورية اليمنية. يتم حل أي نزاعات من خلال محاكم صنعاء.
