Social Analyzer Privacy Policy

🔐 سياسة الخصوصية

📅 آخر تحديث:

نحن في Social Analyzer نلتزم بحماية خصوصيتك وسرية بياناتك. هذه السياسة توضح كيفية تعاملنا مع المعلومات.

1. المعلومات التي يتم تخزينها

لتشغيل خدمات Social Analyzer (@Social_Media_analyzer_bot)، نقوم بتخزين...

  • معرف المستخدم - للتعرف عليك وتقديم الخدمات المخصصة
  • اسم المستخدم والاسم الأول - للترحيب بك وعرضه في صفحة البايو
  • حسابات التواصل الاجتماعي - التي تختار تسجيلها (يوتيوب، تيك توك، سناب شات، انستقرام، فيسبوك)
  • سجل التحليلات - لتتبع تطور حساباتك وتحسين التوصيات

2. استخدام معلوماتك

نستخدم المعلومات المخزنة للأغراض التالية فقط:

  • تقديم خدمة تحليل حسابات التواصل الاجتماعي
  • إنشاء وإدارة صفحة البايو الشخصية الخاصة بك
  • تقديم توصيات ذكاء اصطناعي مخصصة لتحسين قنواتك
  • تحسين أداء البوت وتجربة المستخدم

3. تكامل تيك توك (TikTok API)

🎵 TikTok API Integration

يقدم بوت Social Analyzer (@Social_Media_analyzer_bot) خدمة تحليل حسابات تيك توك من خلال TikTok API الرسمي.

3.1 البيانات التي نحصل عليها

  • البيانات العامة للملف الشخصي (الاسم، الصورة الرمزية، السيرة الذاتية، عدد المتابعين)
  • إحصائيات الأداء (المشاهدات، الإعجابات، التعليقات، المشاركات)
  • قائمة الفيديوهات المنشورة (آخر 10 فيديوهات)

3.2 كيفية الاستخدام

  • إنشاء تقارير تحليلية شاملة
  • تقديم توصيات مخصصة لتحسين الأداء
  • تتبع النمو والتطور بمرور الوقت

3.3 تخزين البيانات وأمانها

  • مدة التخزين: 30 يوماً من آخر تحليل
  • نقل البيانات عبر تشفير HTTPS
  • لا يتم تخزين كلمات المرور أو بيانات تسجيل الدخول

4. تكامل سناب شات (Snapchat Kit)

📸 Snapchat Kit Integration

نوفر ميزة تحليل حسابات سناب شات من خلال Snapchat Kit API الرسمي.

4.1 البيانات التي نحصل عليها

  • البيانات العامة للملف الشخصي (الاسم، المعرف، الصورة الرمزية)
  • إحصائيات الأداء الأساسية (المتابعين، مشاهدات القصص)

5. 🔗 الروابط الخارجية وخدمات فحص اليوزرنيم

📌 تنبيه خاص بخدمة فحص اليوزرنيم (Username Availability Check)

يوفر بوتنا زراً أو رابطاً مختصراً لموقع instantusername.com كخدمة مجانية لتسهيل الوصول إلى أداة فحص توافر اليوزرنيم على منصات متعددة.

  • 🔍 طريقة الفتح: يتم فتح الرابط في المتصفح الداخلي لتطبيق تلجرام (In-App Browser)، وليس عبر خوادم البوت.
  • 🤖 دور البوت محدود: البوت لا يشارك في عملية البحث، ولا يقوم بتحميل النتائج، ولا يخزن أي بيانات تدخلها في الموقع الخارجي.
  • 🌐 استقلالية البيانات: بمجرد فتح الرابط، ينتقل اتصالك مباشرة إلى موقع instantusername.com. جميع البيانات التي تدخلها أو تبحث عنها تصبح خاضعة تماماً لـ سياسة الخصوصية وشروط الخدمة الخاصة بالموقع الخارجي، وليس لسياسة بوتنا.
  • ⚖️ إخلاء مسؤولية: بوت Social Analyzer ومطوروه غير مسؤولين عن توفر الخدمة، دقة النتائج، استمرارية الموقع، أو أي ضرر ناتج عن استخدام موقع instantusername.com. كما أن البوت لا يتحمل أي مسؤولية عن كيفية جمع أو استخدام الموقع الخارجي لبياناتك.
📖 ملخص للشفافية: الرابط يُفتح داخل تلجرام مباشرة (متصفح داخلي) ← أنت من يتفاعل مع الموقع بنفسك ← البوت لا يعرف ماذا تبحث ولا يخزن شيئاً. استخدامك للموقع يعني موافقتك على سياسات instantusername.com.

6. مشاركة المعلومات

نحن لا نبيع أو نشارك معلوماتك الشخصية مع أي طرف ثالث.

7. حماية البيانات

  • تشفير البيانات أثناء النقل والتخزين
  • سياسات التحكم في الوصول (RLS)
  • مراقبة مستمرة للأنظمة

8. صلاحية البيانات

يمكنك في أي وقت: طلب نسخة من بياناتك، حذف بياناتك بالكامل، إلغاء اشتراكك المميز، أو إلغاء صلاحية تطبيقنا على تيك توك/سناب شات.
